webpack作用

webpack本身是, node的一个第三方模块包, 用于打包代码

它是一个流行的前端项目构建工具（打包工具），体狗模块化支持，以及代码压缩混淆，js兼容性处理，性能优化等功能

webpack官网

现代 javascript 应用程序的 静态模块打包器 (module bundler)

把很多文件打包整合到一起, 缩小项目体积, 提高加载速度

左边是没有经过webpack打包之前的项目的状态

其中功能:

  • less/sass -> css

  • ES6/7/8 -> ES5

  • html/css/js -> 压缩合并
